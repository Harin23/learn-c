# Phase 1 — Programming Foundations

Take your first steps in C programming: syntax, types, control flow, functions, arrays, strings, structs, basic debugging, and Git fundamentals.

**Estimated time:** 60–75 hours (5 modules × 12–15 hours)

## Modules

Complete these in order. Each module builds on the previous one.

| Module | Title | Hours |
|--------|-------|-------|
| [1.1](module-01-01-first-c-program/README.md) | First C Program, Variables, and Types | 12–15 |
| [1.2](module-01-02-control-flow/README.md) | Control Flow and Basic I/O | 12–15 |
| [1.3](module-01-03-functions/README.md) | Functions and Program Organization | 12–15 |
| [1.4](module-01-04-arrays-structs/README.md) | Arrays, Strings, Structs, and Enums | 15–18 |
| [1.5](module-01-05-debugging-git/README.md) | Basic Debugging and Git Fundamentals | 12–15 |

## Phase Exit Gate

Do not start Phase 2 until you can independently:

- Write, compile (`gcc -Wall -Wextra -std=c11`), and debug a multi-file C program
- Use structs, arrays, strings, enums, and control flow correctly
- Explain compilation stages (preprocess, compile, assemble, link) at a high level
- Use GDB to set breakpoints, step, print variables, and read backtraces
- Create a Git repo, commit, branch, merge, and write clear commit messages
- Complete the **Phase 1 capstone**: multi-file student gradebook CLI ([project spec](../projects/phase-01/07-gradebook-capstone.md))
